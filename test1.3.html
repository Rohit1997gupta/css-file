E-Library:

This project aims to build a e-library system through which user will be able to login and purchase their choice of book. Users can register themselves to login into the application, users are allowed to search books and purchase the book which they are willing to buy. Library Admin is only allowed to add new book, edit the entire details of the book & delete the book
